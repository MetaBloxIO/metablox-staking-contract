# MetaBlox ERC20 contract

## quick start

``` shell
npm install

npx hardhat compile

npm run test

npx hardhat run scripts/deployMetaBlox.ts

npx hardhat run scripts/deployCappedCashier.ts

```

## hardhat command
```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
npx hardhat help
```
